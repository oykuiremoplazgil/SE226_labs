#  Advanced Programming - Lab Studies

### [TR] Özet
Bu repository, **Yazılım Mühendisliği 2. Sınıf** "İleri Programlama" dersi kapsamında **C++** ve **Python** dilleriyle gerçekleştirdiğim çalışmaları içermektedir. Dersin temel amacı, farklı programlama paradigmalarını kavramak ve her iki dilin güçlü yönlerini kullanarak karmaşık problemleri çözmektir.

**Öne Çıkan Yetkinlikler:**
* **C++:** Bellek yönetimi, pointer kullanımı ve nesne yönelimli programlama (OOP).
* **Python:** Esnek veri yapıları, script yazımı ve modern kütüphane entegrasyonu.
* **Dil Karşılaştırması:** Aynı problemin her iki dildeki performans ve yazım farklarının analizi.

---

### [ENG] Overview
This repository contains lab studies conducted in **C++** and **Python** for the **2nd Year Advanced Programming** course. The primary goal of the course is to master different programming paradigms and solve complex problems by leveraging the strengths of both languages.

**Key Competencies:**
* **C++:** Memory management, pointer manipulation, and Object-Oriented Programming (OOP).
* **Python:** Dynamic data structures, scripting, and modern library integration.
* **Cross-Language Analysis:** Comparing performance and syntax differences for the same problem sets.

---

##  Kullanılan Teknolojiler / Tech Stack
* **Languages:** `C++`, `Python 3.x`
* **Concepts:** `Memory Management`, `OOP`, `File Handling`, `Algorithm Implementation`.
